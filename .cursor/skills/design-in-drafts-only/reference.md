# 设计仅草案 · 速查

## 草案目录

- **路径**：`docs/design/drafts/`
- **说明**：`docs/design/drafts/README.md`

## 允许改动的范围（设计阶段）

- 仅 `docs/design/drafts/*.md`：新建或编辑草稿，命名见该目录 README。
- 草稿文末须有「最后更新：YYYY-MM-DD HH:MM」（精确到分钟），每次修改草稿时更新。

## 禁止改动的范围（设计阶段）

- `plugins/`、`main.py`、`db/`、`config.yaml` 及任何实现代码与运行配置。

## 视为「可以开始写代码」的用户表述

- 开始执行、开始实现、可以写代码了、去实现吧
- 修改代码、编写代码、写代码、实现这个设计
- 按草稿实现、按设计稿开发

未出现上述或类似表述时，一律只改草案，不改代码。
