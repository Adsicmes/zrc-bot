# 拟设计草稿

存放**插件或功能的拟设计草稿**，便于讨论和迭代，不要求与实现完全一致。

## 命名建议

- 按插件：`插件名.md`（如 `my_feature_plugin.md`）
- 按主题：`主题-日期.md`（如 `权限扩展-2025-02.md`）
- 按功能点：`功能简述.md`

## 草稿可包含

- 目标与背景
- 命令/接口设想
- 权限与配置
- 与现有插件的关系
- 待定问题（TBD）
- **最后更新时间**：文末注明 `最后更新：YYYY-MM-DD HH:MM`（精确到分钟），每次修改草稿时更新。

定稿后请更新 [设计考虑](../design.md) 或 [插件文档](../../plugins/README.md) 中的对应部分。
