# 牛牛大作战 · 使用说明

在群里用「牛牛长度」玩轻量娱乐：发指令随机增减长度，还能查自己/别人、看排行榜。**仅群聊可用**，且需要管理员先在本群发送「开启牛牛大作战」才会生效。

---

## 基本规则

- 每人**初始长度 8.00 cm**，首次参与时自动按这个值。
- 通过「导」「日」等指令会**随机增减**长度（单次约 ±2 cm 内）。
- 每个群的数据**独立**，同一人在不同群有不同长度。
- 所有指令**不需要加斜杠**，直接发文字即可。

---

## 指令说明与示例

### 改变自己的长度：导

| 说明 | 发送后自己的牛牛长度随机变化一次，当天「设了」次数 +1。 |
|------|--------------------------------------------------------|
| **指令** | 导 |
| **示例** | `导` |

---

### 改变别人的长度：日群友 / 草群友 / 操群友

| 说明 | 对别人的牛牛长度造成一次随机变化。单独发则随机选一名群友；@ 某人则指定对象。**不能 @ 自己**，想改自己的用「导」。 |
|------|--------------------------------------------------------------------------------------------------------------|
| **指令** | 日群友 / 草群友 / 操群友（任选一种说法）；可带 @ 指定对象 |
| **示例** | `日群友`<br>`日群友 @张三`<br>`草群友`<br>`操群友 @李四` |

---

### 查自己的长度：我的牛牛 / 我的牛子

| 说明 | 查看自己在本群的当前牛牛长度。 |
|------|------------------------------|
| **指令** | 我的牛牛 / 我的牛子（任选一种） |
| **示例** | `我的牛牛`<br>`我的牛子` |

---

### 查别人的长度：查看牛牛@某人

| 说明 | 查看被 @ 用户在本群的牛牛长度。**必须 @ 一个人**。 |
|------|--------------------------------------------------|
| **指令** | 查看牛牛 + @ 要查的人 |
| **示例** | `查看牛牛 @张三` |

---

### 本群排行榜：牛牛排行榜

| 说明 | 显示本群牛牛长度**前十名（最长）**和**后十名（最短）**。 |
|------|--------------------------------------------------------|
| **指令** | 牛牛排行榜 |
| **示例** | `牛牛排行榜` |

---

## 管理员专用指令

以下指令需要**管理员权限**（Bot 的 admin/root）。

### 开启 / 关闭本群牛牛大作战

| 说明 | 在当前群发送「开启牛牛大作战」后，本群才能使用上面所有指令；发送「关闭牛牛大作战」后本群不再响应。 |
|------|--------------------------------------------------------------------------------------------------|
| **指令** | 开启牛牛大作战 / 关闭牛牛大作战 |
| **示例** | `开启牛牛大作战`<br>`关闭牛牛大作战` |

### 设置某人的数据：设置牛牛@某人 键 数值

| 说明 | 修改被 @ 用户在本群的「长度」或「设了次数」。键只能填 **长度** 或 **设了次数**，数值按规则填写（长度支持小数，设了次数为非负整数）。 |
|------|------------------------------------------------------------------------------------------------------------------------------|
| **指令** | 设置牛牛 @某人 键 数值 |
| **示例** | `设置牛牛 @张三 长度 10.50`<br>`设置牛牛 @李四 设了次数 0` |

---

## 小提示

- 想改自己的长度时请用「**导**」，不要用「日」@ 自己，会提示你用导。
- 第一次参与或从未被查过的人，显示的长度为初始值 **8.00 cm**。
- 本群没人参与过时，发「牛牛排行榜」会提示暂无数据。
